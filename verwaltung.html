Wo bekomme ich den Zugangsschlüssel her? (einmalig, 2 Minuten)
  1. github.com → Einstellungen → Fine-grained token öffnen (mit dem Galerie-Konto anmelden)
  2. Name z. B. „Galerie-Verwaltung", Ablauf 1 Jahr, Only select repositories → galerie-jp-showroom
  3. Unter „Repository permissions" bei Contents „Read and write" wählen → „Generate token"
  4. Den angezeigten Schlüssel oben einfügen — fertig. Er bleibt nur in diesem Browser gespeichert.
